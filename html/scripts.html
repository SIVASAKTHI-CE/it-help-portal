📜 Automation Scripts

Download and run pre-built shell scripts for one-command setup of security tools and dev environments.

LINUXBASH
security_setup.sh
Installs CrowdStrike Falcon Sensor on Ubuntu. Sets Customer ID (CID), enables and starts the falcon-sensor service automatically. Run once on any new Ubuntu machine.
$ chmod +x security_setup.sh
$ sudo bash security_setup.sh
# Verify: sudo systemctl status falcon-sensor

mobility_setup.sh
Installs and configures Zscaler Private Access client on Ubuntu. Includes OpenSSH server setup, UFW firewall rules, and service auto-start configuration.
$ chmod +x mobility_setup.sh
$ sudo bash mobility_setup.sh
# Verify: sudo systemctl status zscaler

ror_setup.sh
Full Ruby on Rails development environment setup. Installs RVM, Ruby 3.2.2, Rails 7.0, MySQL server and MySQL Workbench. One script does everything.
$ chmod +x ror_setup.sh
$ bash ror_setup.sh
# Verify: ruby -v && rails -v

bios_update.sh
Automates BIOS/firmware update checks for Dell systems using fwupdmgr. Safe to run on any Ubuntu machine connected to the internet.
$ chmod +x bios_update.sh
$ sudo bash bios_update.sh
# Requires: fwupdmgr installed
